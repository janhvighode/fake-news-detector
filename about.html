About This AI System

This project uses Artificial Intelligence and Natural Language Processing to detect whether a news article is real or fake. The system analyzes the text patterns and compares them with a trained dataset.

Technologies Used

  • Python
  • Flask Web Framework
  • Scikit-learn
  • Natural Language Processing (NLP)
  • TF-IDF Feature Extraction

Model Performance

  • Algorithm: Logistic Regression
  • Text Processing: TF-IDF Vectorization
  • Dataset Size: 40,000+ news articles
  • Training Split: 80% Training / 20% Testing
  • Accuracy: ~95%

The model was trained on a large dataset of real and fake news articles, allowing it to detect misinformation with high accuracy.
